INSTALLATION INSTRUCTIONS

1. UNZIP FILE TO DESTINATION OF YOUR CHOICE

2. DOWNLOAD PYTHON 3.8

3. INSTALL PYGAME
    3.a OPEN CMD
    3.b enter "pip install pygame" without quotations

4. RUN ELECTRICITY.BAT

OPTIONAL: CREATE A SHORTCUT

5. RIGHT CLICK DESKTOP > NEW > SHORTCUT

6. BROWSE TO FIND THE INSTALLATION LOCATION AND CLICK ELECTRICITY.BAT



CONTROLS

    MENU

        CLICK ON A PLAYER TO CHANGE THE NAME

        PRESS ENTER TO CONFIRM

    GAME

        SPACEBAR - DRAWS A NEW CARD

        T - STARTS TIMER FROM THE HIGHEST CURRENT CONNECTED VALUE

        Q - QUITS GAME
